Traffic Light Project

Project Overview

This project demonstrates the use of a Raspberry Pi 4b to control a traffic light system. Threading in Python is used to manage multiple tasks concurrently.

Project Features

  • Pedestrian button to trigger pedestrian lights
  • Left turn signal with each cycle
  • Real-time terminal updates
  • GPIOZero library for Raspberry Pi

Project Explanation

Simulates real-world traffic lights with pedestrian controls. Green, yellow, and red phases adjust based on button presses.
