Applied Linear Regression on the Auto Data Set

Project Overview

This project demonstrates both simple and multiple linear regression techniques using Python and the Auto dataset. We analyzed how vehicle characteristics affect fuel efficiency (mpg) and assessed model quality using diagnostic plots.

Notebook with full code: View on GitHub

Task A: Simple Linear Regression (Horsepower → MPG)

Code

import pandas as pd
import statsmodels.api as sm
from statsmodels.sandbox.regression.predstd import wls_prediction_std
import matplotlib.pyplot as plt

# Load Auto dataset
Auto = pd.read_csv('Auto.csv', na_values='?').dropna()
X = sm.add_constant(Auto['horsepower'])
y = Auto['mpg']

# Fit simple linear regression
model = sm.OLS(y, X).fit()
print(model.summary())

# Prediction for horsepower = 98
hp_value = 98
pred = model.get_prediction([1, hp_value])
print(pred.summary_frame(alpha=0.05))

# Plot
fig, ax = plt.subplots()
ax.scatter(Auto['horsepower'], Auto['mpg'], color='blue', label='Data')
ax.axline([0, model.params[0]], slope=model.params[1], color='red', label='OLS line')
ax.set_xlabel('Horsepower')
ax.set_ylabel('MPG')
ax.set_title('Simple Linear Regression: MPG vs Horsepower')
ax.legend()
plt.show()

Results

Summary of findings:

  • Relationship: Horsepower negatively impacts mpg.
  • Strength: Moderately strong (R-squared ≈ 0.605).
  • Predicted mpg for horsepower = 98: 24.47
  • 95% Confidence Interval: [23.0, 25.9]
  • 95% Prediction Interval: [15.3, 33.6]

Diagnostic Plots

The residuals show slight heteroscedasticity; most observations fall within expected variance, but a few outliers exist.

Task B: Multiple Linear Regression (All Predictors)

Code

import seaborn as sns
import numpy as np
from statsmodels.stats.anova import anova_lm

# Scatterplot matrix
sns.pairplot(Auto)
plt.show()

# Correlation matrix
corr_matrix = Auto.corr()
print(corr_matrix)

# Multiple regression
predictors = Auto.columns.drop(['mpg', 'name'])
X_multi = sm.add_constant(Auto[predictors])
y = Auto['mpg']
model_multi = sm.OLS(y, X_multi).fit()
print(model_multi.summary())

# ANOVA
anova_results = anova_lm(sm.OLS(y, sm.add_constant(np.ones(len(y)))).fit(), model_multi)
print(anova_results)

# Interaction & transformations
# Example: mpg ~ horsepower * weight
model_interact = sm.OLS(y, sm.add_constant(Auto[['horsepower','weight']]))
model_interact = model_interact.fit()
print(model_interact.summary())

Results

Key findings:

  • Overall, predictors significantly explain variation in mpg (p < 0.001 from ANOVA).
  • Significant predictors: weight, year, horsepower, displacement.
  • Year coefficient: positive — newer cars have higher mpg.
  • Diagnostics: residuals mostly normal; a few high-leverage points exist.
  • Interactions & transformations (e.g., horsepower * weight, log(displacement)) slightly improved model fit.

Scatterplot & Correlation

The scatterplot matrix shows relationships among variables, and the correlation table confirms multicollinearity between some predictors.
