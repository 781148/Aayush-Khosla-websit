Task A: Simple Linear Regression (Question 8)

(a) Regression Analysis

import statsmodels.api as sm
X = sm.add_constant(Auto['horsepower'])
y = Auto['mpg']
model = sm.OLS(y, X).fit()
print(model.summary())

Interpretation:

  • There is a strong negative relationship between horsepower and mpg.
  • The model explains a significant portion of variation in mpg (R² ≈ 0.605).
  • Higher horsepower is associated with lower mpg (negative slope).
  • Predicted mpg for 98 horsepower: ≈ 24.47
  • 95% Confidence Interval: [23.51, 25.43]
  • 95% Prediction Interval: [15.69, 33.25]

Figure: Scatter plot of mpg vs horsepower with least squares regression line.

(c) Diagnostic Plots

Figure: Residual plots showing variance and potential mild non-linearity.

Commentary: Residuals show some slight pattern, suggesting non-linearity may exist for extreme horsepower values. No extreme outliers are apparent.

Task B: Multiple Linear Regression (Question 9)

(a) Scatterplot Matrix

Figure: Scatterplot matrix for all variables in the Auto dataset.

(b) Correlation Matrix

correlations = Auto.corr()

Commentary: Strong negative correlation between horsepower and mpg; other predictors like weight and displacement also show negative correlations with mpg.

(c) Multiple Linear Regression

predictors = ['cylinders','displacement','horsepower','weight','acceleration','year','origin']
X_multi = sm.add_constant(Auto[predictors])
y = Auto['mpg']
model_multi = sm.OLS(y, X_multi).fit()
print(model_multi.summary())

Interpretation:

  • The predictors together significantly explain variation in mpg (p < 0.001).
  • Significant predictors: cylinders, displacement, horsepower, weight, year.
  • The coefficient for year is positive, suggesting newer cars have higher mpg.

(d) Diagnostic Plots

Figure: Residuals and leverage plots for multiple regression. Some observations have higher leverage but no extreme outliers.

(e) Interaction Models

# Example interaction term
X_interact = X_multi.copy()
X_interact['horsepower_weight'] = X_interact['horsepower'] * X_interact['weight']
model_interact = sm.OLS(y, X_interact).fit()
print(model_interact.summary())

Commentary: Interaction between horsepower and weight is statistically significant (p < 0.05), indicating combined effect impacts mpg more than individually.
