Linear Regression Tutorial

I worked through this linear regression tutorial using Python. Below are my notes and answers to the questions in the notebook.

Key Questions from the Notebook

1. What is `m` in y = mx + b?

`m` is the coefficient or slope of the line. It tells us how much y changes for a unit change in x.

2. What is the R squared?

R squared measures how well the regression line fits the data. A value closer to 1 means the line explains most of the variance in y.

3. What is the Adj. R squared?

Adjusted R squared accounts for the number of predictors in the model. It adjusts the R squared value so it doesn’t artificially increase when more variables are added.

4. What is the P value?

The P value tells us whether the slope of the line is statistically significant. A small P value (typically ≤ 0.05) means the slope is significantly different from zero.

5. What are the 95% confidence intervals?

The 95% confidence intervals give the range in which the true slope and intercept values are expected to fall 95% of the time.
