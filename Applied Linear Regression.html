Applied Linear Regression on the Auto Dataset

View Full Jupyter Notebook on GitHub

Introduction

This project applies linear regression techniques to the Auto dataset to explore how vehicle characteristics impact fuel efficiency (mpg). Simple and multiple regression models were used, with visualizations and diagnostic checks, to interpret the results.

Simple Linear Regression: MPG vs Horsepower

We used horsepower as the predictor and mpg as the response for a simple linear regression model.

The regression indicates a strong negative relationship: as horsepower increases, fuel efficiency decreases.
Figure 1: Scatter plot of MPG vs Horsepower with fitted regression line.

The model predicts mpg for a vehicle with 98 horsepower, including the 95% confidence and prediction intervals. This allows estimation of fuel efficiency along with expected variability.

Multiple Linear Regression: Predicting MPG

All numeric predictors except the car name were included in a multiple linear regression model to predict mpg. A scatterplot matrix and correlation matrix helped understand variable relationships.

Figure 2: Predicted vs Actual MPG from the multiple regression model.

Regression coefficients reveal the effect of each variable on mpg. For example, the 'year' coefficient shows that newer cars generally have better fuel efficiency. Diagnostic plots help identify outliers and influential observations.

Reflections and Insights

  • Understanding predictor-response relationships is crucial before modeling.
  • Regression coefficients should be interpreted in context of the variables.
  • Diagnostic checks ensure model assumptions are valid.
  • Transformations or interactions can improve model performance.
