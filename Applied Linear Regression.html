Applied Linear Regression: Analysis of the Auto Dataset

This project demonstrates simple and multiple linear regression techniques using Python to explore the relationship between vehicle characteristics and fuel efficiency (mpg).

Task A: Simple Linear Regression (mpg ~ horsepower)

Regression Summary

import statsmodels.api as sm
X = sm.add_constant(Auto['horsepower'])
y = Auto['mpg']
model = sm.OLS(y, X).fit()
print(model.summary())

The regression coefficient for horsepower is -0.157, indicating that higher horsepower reduces fuel efficiency. R² = 0.605 shows a moderate fit.

Prediction for Horsepower = 98

new_hp = pd.DataFrame({'const':1,'horsepower':[98]})
pred = model.get_prediction(new_hp)
pred_summary = pred.summary_frame(alpha=0.05)
print(pred_summary)

The predicted mpg for a car with 98 horsepower is 24.47 mpg with a 95% confidence interval of [23.05, 25.89] and a 95% prediction interval of [16.23, 32.71].

Scatterplot with Regression Line

import matplotlib.pyplot as plt
fig, ax = plt.subplots()
ax.scatter(Auto['horsepower'], Auto['mpg'])
ax.plot(Auto['horsepower'], model.predict(X), color='red', linewidth=2)
ax.set_xlabel('Horsepower')
ax.set_ylabel('MPG')
ax.set_title('MPG vs Horsepower with Regression Line')
plt.show()

Figure: Scatterplot shows a negative relationship between horsepower and mpg.

Diagnostic Plots & Observations

sm.graphics.plot_regress_exog(model, 'horsepower')
plt.show()

Residuals vs fitted values shows slight heteroscedasticity. Q-Q plot indicates residuals are approximately normal, no extreme outliers observed.

Task B: Multiple Linear Regression

Scatterplot Matrix

import seaborn as sns
sns.pairplot(Auto)
plt.show()

Shows relationships among all numeric variables. Some strong correlations are visible, like between displacement and horsepower.

Correlation Matrix

corr_matrix = Auto.corr()
print(corr_matrix)

Confirms numeric relationships between variables. Example: mpg negatively correlated with horsepower (-0.78).

Multiple Regression Analysis

predictors = Auto.columns.drop(['mpg','name'])
X_multi = sm.add_constant(Auto[predictors])
y = Auto['mpg']
model_multi = sm.OLS(y, X_multi).fit()
print(model_multi.summary())

ANOVA confirms significant relationship between predictors and mpg. Significant predictors include horsepower, weight, year. Year coefficient positive: newer cars tend to have higher mpg.

Diagnostic Plots

sm.graphics.plot_partregress_grid(model_multi)
plt.show()

Residuals and leverage plots show no extreme outliers, though a few points have high leverage.

Interactions & Transformations

# Example: Interaction between horsepower and weight
Auto['hp_weight'] = Auto['horsepower'] * Auto['weight']
X_inter = sm.add_constant(Auto[['horsepower','weight','hp_weight']])
model_inter = sm.OLS(y, X_inter).fit()
print(model_inter.summary())

Interaction between horsepower and weight is significant. Transformations like log(weight) or sqrt(horsepower) explored; some improve fit slightly.
