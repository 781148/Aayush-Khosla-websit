Applied Linear Regression Project

Project Overview

This project applies simple and multiple linear regression on the Auto dataset to analyze how vehicle characteristics affect fuel efficiency (mpg).

Simple Linear Regression

Response variable: mpg
Predictor: horsepower

# Python code snippet
X = sm.add_constant(Auto['horsepower'])
model_simple = sm.OLS(Auto['mpg'], X).fit()
print(model_simple.summary())

We observed a negative relationship between horsepower and mpg. Higher horsepower cars generally have lower fuel efficiency.

Multiple Linear Regression

Response variable: mpg
Predictors: cylinders, displacement, horsepower, weight, acceleration, year, origin dummies

# Fit multiple regression
X_multi = sm.add_constant(Auto2[predictors]).astype(float)

The multiple regression model explains more variation in mpg. Significant predictors include weight, year, and horsepower.

Interactions and Transformations

# Interaction example
formula_inter = 'mpg ~ horsepower * weight + displacement + acceleration + cylinders + year'
model_inter = smf.ols(formula_inter, data=Auto).fit()
print(model_inter.summary())

# Transformations
Auto_trans['log_weight'] = np.log(Auto_trans['weight'])
Auto_trans['sqrt_disp'] = np.sqrt(Auto_trans['displacement'])
formula_trans = 'mpg ~ horsepower + log_weight + sqrt_disp + acceleration + cylinders + year'
model_trans = smf.ols(formula_trans, data=Auto_trans).fit()
print(model_trans.summary())

Exploring interactions and transformations improved model fit. The log(weight) transformation, for example, increased interpretability.

Conclusion

Simple regression highlighted the negative effect of horsepower on mpg. Multiple regression incorporated multiple vehicle factors, showing weight, year, and horsepower as key influencers. Interactions and variable transformations further refined model predictions, making this a robust analysis of the Auto dataset.
