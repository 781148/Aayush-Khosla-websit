Visualizing Anscombe’s Quartet

Project Overview

This project demonstrates exploratory data analysis (EDA) on Anscombe’s Quartet using Python. It highlights why visual inspection is essential even when summary statistics are nearly identical.

Project Features

  • Scatter plots with regression lines for all datasets
  • Residual plots
  • Box and violin plots for distributions
  • Interactive versions (Plotly / Altair)
