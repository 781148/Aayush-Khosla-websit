Applied Linear Regression: Auto Dataset

Introduction

This project analyzes the Auto dataset to explore relationships between car characteristics and fuel efficiency (mpg) using simple and multiple linear regression. All analysis was done in Python with statsmodels and pandas, with visualization to interpret model results and detect potential issues.

Simple Linear Regression

We start by analyzing the relationship between horsepower and mpg. The model uses mpg as the response variable and horsepower as the predictor.

Model Output

The regression results indicate a negative relationship between horsepower and mpg. Higher horsepower is associated with lower fuel efficiency. The coefficient for horsepower is statistically significant, indicating that it reliably predicts mpg.

Figure 1: Scatterplot of mpg vs. horsepower with the fitted regression line. The line shows the predicted mpg based on horsepower.

Insight: Cars with higher horsepower tend to have lower mpg, which aligns with expectations that more powerful engines consume more fuel.

Multiple Linear Regression

We then expanded to multiple linear regression using all numeric predictors (cylinders, displacement, horsepower, weight, acceleration, year, origin) to predict mpg. This allows us to account for the combined effects of multiple variables.

Model Output & Analysis

The results show several predictors are statistically significant, including:

  • Horsepower: negative effect on mpg
  • Weight: heavier cars have lower mpg
  • Year: newer cars tend to have higher mpg

Figure 2: Residual plots for the multiple regression model. These help assess the assumptions of linearity, homoscedasticity, and identify potential outliers.

Insight: While most residuals are randomly distributed, a few high-leverage points suggest unusual cars that may influence the model fit.

Reflections & Learnings

This project reinforced the importance of:

  • Exploratory data analysis before modeling
  • Checking model assumptions using diagnostic plots
  • Interpreting regression coefficients in context
  • Considering multiple predictors together for more accurate predictions

The combination of visualizations and regression results provides a complete picture of how car characteristics relate to fuel efficiency.
